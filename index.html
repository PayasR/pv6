The Design of pv6

Background

pv6 is a fork of the teaching operating system xv6, which is a Unix-like OS based on Sixth Edition UNIX. The operating system runs on multiprocessor x86 machines. Pv6 is being developed through the Google Summer of Code program with the support of the lowRISC organization. Pv6 started as an attempt to port xv6 to the RISC-V architecture in general and the lowRISC platform in particular, however, at the suggestion of Alex Bradbury, my GSoC mentor for the project, we decided that it would be useful and only slightly more work to restructure the source code of xv6 to make it easy to port to other architectures as well. That said, my work this summer will only focus on porting to RISC-V.

This webpage collects the documentation I write explaining the design and rationale of the RISC-V port. I hope that it is helpful for those curious about operating system implementation or who want to port pv6 to other architectures. As the summer progresses and I write up more of the development process I will continue to update the list below.
